## HyperFlux

HyperFlux is a fullscreen starfield simulation built with Pygame that recreates a warp-speed space effect in real time. It generates thousands of stars in a 3D space and projects them onto the screen, creating the illusion of rapid forward motion as they accelerate toward the viewer. The animation is smooth and immersive, enhanced by motion blur and dynamic brightness based on depth. Users can change the color of the stars instantly using keyboard input, making the experience both interactive and visually engaging.
